Git – это распределённая система контроля версий, позволяющая отслеживать изменения
в любых файлах, часто используемая для координации работы программистов, совместно
разрабатывающих исходный код во время разработки проекта.
Git был создан Линусом Торвальдсом в 2005 году для разработки ядра Linux.
На сегодняшний день его поддерживает Джунио Хамано.
